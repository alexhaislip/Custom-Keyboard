# Custom-Keyboard

## Introduction

The QWERTY keyboard is the most common keyboard around the world. \
However, it is not the most efficient keyboard. \
If you use this keyboard a lot, you may strain your fingers later. \
This project aims to find a better keyboard using genetic algorithms.

![image](https://user-images.githubusercontent.com/87184009/135722399-8c066fc0-9978-438d-9447-cc105d614ff3.png)

## Usage

Put all the documents you wrote in the text data folder and run main.py.
